# 사이트 오픈 완전 초보자 가이드

이 문서는 컴퓨터/개발 지식이 전혀 없어도 순서대로 따라 하면 사이트를 실제로 인터넷에 열고, 돈을 받고, 광고를 달고, 홍보까지 할 수 있도록 쓴 안내서입니다. 위에서부터 순서대로 진행하세요. 각 단계는 이전 단계가 끝나야 다음 단계로 넘어갈 수 있습니다.

전체 순서를 먼저 보여드립니다:

1. GitHub(코드 보관 서비스) 가입 + 코드 올리기
2. Render(호스팅 서비스) 가입 + 사이트 배포 (여기서부터 인터넷 주소가 생김)
3. 도메인(www.내사이트.com 같은 주소) 구매 + 연결
4. 개인사업자 등록 (실제 결제를 받으려면 필수)
5. 토스페이먼츠 실결제 전환
6. 구글 애드센스 신청 (광고 수익)
7. AI 자동관리 에이전트 활성화
8. 홍보 방법

---

## 0. 시작 전에 알아둘 것

- **비용**: 도메인 1년 약 1~2만원, 호스팅은 처음엔 무료 플랜으로 충분, 사업자등록은 무료, 애드센스/토스 심사도 무료입니다. 초기 비용은 도메인값 정도로 시작할 수 있습니다.
- **시간**: 1~2단계(코드 올리기, 배포)는 오늘 30분~1시간이면 끝납니다. 4~6단계(사업자등록, 결제 심사, 애드센스 심사)는 각각 며칠씩 걸릴 수 있으니 미리 신청해두고 기다리는 것이 좋습니다.
- 막히는 부분이 있으면 이 대화로 돌아와서 "○단계에서 막혔어"라고 말씀해주시면 도와드립니다.

## 1. GitHub 가입 + 코드 올리기

GitHub는 여러분의 사이트 코드를 안전하게 보관하고, 다음 단계인 Render 호스팅이 그 코드를 가져다 쓸 수 있게 해주는 서비스입니다. 프로그래밍 지식 없이도 **GitHub Desktop**이라는 프로그램(마우스 클릭만으로 사용 가능)을 쓰면 됩니다.

1. https://github.com 접속 → 오른쪽 위 **Sign up** 클릭 → 이메일, 비밀번호, 사용자명을 입력해 무료 가입
2. https://desktop.github.com 접속 → **Download for Windows** 클릭해서 설치
3. GitHub Desktop을 실행 → 방금 만든 GitHub 계정으로 로그인
4. 왼쪽 위 **File → Add Local Repository** 클릭
5. **Choose...** 버튼을 눌러 `C:\Users\user\Desktop\새 폴더`를 선택 → **Add Repository** 클릭
   - (이미 이 폴더는 코드 준비가 되어 있고 git이 초기화되어 있어서 바로 인식됩니다)
6. 화면 왼쪽에 변경된 파일 목록이 보이면, 아래쪽 **Summary** 칸에 "첫 업로드"라고 입력하고 **Commit to master** 버튼 클릭
7. 화면 위쪽 **Publish repository** 버튼 클릭 → Name은 원하는 이름(예: stock-recommender)으로 → **Keep this code private** 체크는 해제해도 되고 두어도 됩니다 (Render 무료 플랜은 비공개 저장소도 연결 가능) → **Publish Repository** 클릭

여기까지 하면 코드가 GitHub에 올라갑니다. 앞으로 사이트 코드를 수정할 때마다 GitHub Desktop에서 **Commit** → **Push origin** 버튼만 누르면 자동으로 반영됩니다.

## 2. Render로 사이트 배포하기

Render는 여러분의 컴퓨터가 꺼져 있어도 사이트가 24시간 인터넷에 떠 있게 해주는 무료 호스팅 서비스입니다.

1. https://render.com 접속 → **Get Started** → **GitHub으로 계속하기**를 선택해 방금 만든 GitHub 계정으로 가입/로그인
2. 대시보드에서 **New +** 버튼 → **Web Service** 선택
3. 방금 GitHub에 올린 저장소(예: stock-recommender)를 목록에서 찾아 **Connect** 클릭
4. 설정 화면에서 아래처럼 입력:
   - **Name**: 원하는 이름 (예: stock-recommender)
   - **Region**: Singapore (한국과 가장 가까움)
   - **Root Directory**: `backend`
   - **Runtime**: Python 3
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `uvicorn app.main:app --host 0.0.0.0 --port $PORT`
   - **Instance Type**: Free
5. 아래로 스크롤해 **Environment Variables**(환경변수) 섹션에서 **Add Environment Variable**을 눌러 아래 항목들을 하나씩 추가:
   - `ADMIN_TOKEN` = 아무 임의의 긴 문자열 (예: 직접 만든 비밀번호처럼, 예: `sk9x2m8q...`처럼 아무도 못 맞출 값). 이 값은 나중에 AI 자동관리 에이전트를 연결할 때도 똑같이 사용합니다.
   - (5단계에서 발급받을 `TOSS_CLIENT_KEY`, `TOSS_SECRET_KEY`는 지금은 비워두고 나중에 추가해도 됩니다)
6. **Create Web Service** 클릭 → 몇 분 기다리면 빌드/배포가 끝나고, 화면 위쪽에 `https://stock-recommender-xxxx.onrender.com` 같은 주소가 생깁니다. 이 주소를 클릭해 사이트가 뜨는지 확인하세요.

**참고**: 무료 플랜은 15분 동안 방문자가 없으면 서버가 잠들었다가, 다음 방문 시 15~30초 정도 깨어나는 데 시간이 걸립니다. 방문자가 늘어나면 유료 플랜(월 7달러 수준)으로 올리는 것을 권장합니다.

## 3. 도메인 구매 + 연결

지금 주소(`....onrender.com`)도 작동은 하지만, `내사이트.com`처럼 짧고 기억하기 쉬운 주소를 쓰고 싶다면 도메인을 구매하세요.

1. https://www.gabia.com (가비아, 한국어 지원) 접속 → 회원가입
2. 상단 **도메인** 메뉴 → 원하는 이름 검색 (예: mystockapp.com) → 사용 가능하면 장바구니에 담아 결제 (연 1~2만원)
3. 구매 후 가비아 **My가비아 → 서비스 관리 → 도메인 관리**로 이동 → 구매한 도메인 클릭 → **네임서버/DNS 설정** 메뉴로 이동
4. 아래 레코드를 추가:
   - 타입 `CNAME`, 호스트 `www`, 값 `stock-recommender-xxxx.onrender.com` (2단계에서 받은 Render 주소)
5. Render 대시보드로 돌아가 해당 서비스 → **Settings → Custom Domains → Add Custom Domain** → `www.내도메인.com` 입력 → 안내에 따라 인증 (Render가 자동으로 무료 HTTPS 보안인증서도 붙여줍니다)
6. 반영까지 몇 분~몇 시간 걸릴 수 있습니다. 다 되면 `https://www.내도메인.com`으로 사이트가 열립니다.

## 4. 개인사업자 등록 (실결제를 받기 위해 필수)

실제 돈으로 결제를 받으려면(4단계) 사업자등록번호가 있어야 합니다. 온라인으로 무료·당일 처리가 가능합니다.

1. https://www.hometax.go.kr (국세청 홈택스) 접속 → 회원가입(공동인증서 또는 간편인증 필요 — 은행 앱이나 카카오/네이버 인증서로 가능)
2. 로그인 후 상단 메뉴 **국세증명·사업자등록·세금관련 신청/신고 → 사업자등록 신청/정정 → 사업자등록 신청(개인)** 클릭
3. 기본 정보 입력 후 **업종 입력** 단계에서 업종코드 검색창에 "정보통신업" 또는 "온라인 정보제공업"으로 검색해 가까운 업종 선택 (콘텐츠 제공 + 광고수익 구조에 해당)
4. 사업장 주소는 자택 주소로 등록 가능합니다 (별도 사무실 불필요)
5. 제출 후 보통 당일~2일 이내 **사업자등록증**이 홈택스에서 발급됩니다 (마이페이지에서 PDF로 다운로드 가능)
6. **통신판매업 신고**: 온라인으로 유료 서비스를 판매하므로 통신판매업 신고도 필요합니다. https://www.gov.kr (정부24) 접속 → "통신판매업 신고" 검색 → 온라인 신청 (사업자등록증, 구매안전서비스 이용 확인증이 필요한데, 이 확인증은 토스페이먼츠 심사 신청 시 발급받을 수 있으니 5단계를 먼저 진행해도 됩니다)

## 5. 토스페이먼츠 실결제 전환

지금 사이트는 테스트 결제만 가능합니다. 실제 돈을 받으려면 심사를 통과해 라이브(실서비스) 키를 받아야 합니다.

1. https://developers.tosspayments.com 접속 → 가입 (이미 4단계에서 만든 계정이 없다면 새로 가입)
2. 로그인 후 **내 상점** 또는 **가맹점 심사 신청** 메뉴로 이동해 신청서 작성
   - 사업자등록증 사본 업로드 (4단계에서 발급받은 것)
   - 정산받을 본인 명의 은행 계좌 등록
   - 사이트 주소 입력 (3단계에서 만든 도메인 주소)
   - 서비스 소개: "국내 주식 종목 정보 및 유사종목 추천 콘텐츠 유료 제공"처럼 간단히 작성
3. 심사는 보통 영업일 기준 2~5일 소요됩니다. 심사 중 서류 보완 요청이 오면 안내에 따라 제출하세요.
4. 승인되면 개발자센터 **API 키** 메뉴에서 **라이브 클라이언트 키**(`live_ck_...`)와 **라이브 시크릿 키**(`live_sk_...`)를 확인할 수 있습니다.
5. Render 대시보드 → 해당 서비스 → **Environment** 탭에서:
   - `TOSS_CLIENT_KEY` = 방금 받은 `live_ck_...` 값으로 설정
   - `TOSS_SECRET_KEY` = 방금 받은 `live_sk_...` 값으로 설정
   - 저장하면 Render가 자동으로 재배포하며 반영됩니다 (코드 수정 불필요)
6. 이제부터 사이트에서 결제하면 실제 돈이 오갑니다. 소액(1,000원 등)으로 본인이 먼저 테스트 결제를 해보는 것을 권장합니다.

## 6. 구글 애드센스 신청 (광고 수익)

1. 신청 전 체크리스트 (이미 이 프로젝트에 준비되어 있습니다):
   - ✅ 개인정보처리방침 페이지 (`/static/privacy.html`)
   - ✅ 이용약관 페이지 (`/static/terms.html`)
   - ✅ 소개/문의 페이지 (`/static/about.html`) — **여기 안의 이메일 주소를 본인 실제 이메일로 바꿔주세요** (`backend/static/about.html` 파일에서 `contact@example.com` 부분)
   - ✅ 오늘의 시황 콘텐츠 (7단계에서 자동으로 쌓입니다 — 최소 1~2주는 콘텐츠가 쌓인 뒤 신청하는 것을 권장합니다)
2. https://www.google.com/adsense 접속 → **시작하기** → 구글 계정으로 로그인
3. 사이트 주소(3단계에서 만든 도메인)를 입력하고 안내에 따라 등록
4. 애드센스가 발급하는 코드 스니펫을 `backend/static/index.html`의 `<head>` 안에 붙여넣어야 하는데, 이 부분은 Claude에게 "애드센스 코드 붙여줘, 코드는 [코드 내용]이야"라고 말씀해주시면 대신 넣어드립니다
5. 심사는 보통 몇 일~몇 주가 걸립니다. 통과하면 이메일로 알림이 오고, 이후 승인된 자리에 광고가 자동으로 표시됩니다.
6. **주의**: 심사 통과 전까지는 스스로 광고를 클릭하거나 지인에게 클릭을 부탁하면 안 됩니다 (계정 정지 사유).

## 7. AI 자동관리 에이전트 활성화

이미 3개의 자동 관리 에이전트가 예약되어 있습니다 (Claude 앱의 "Scheduled" 메뉴에서 확인 가능):
- **데이터 갱신 에이전트**: 평일 저녁 6시 50분, 종목 데이터 최신화
- **장애 감시 에이전트**: 매시간, 사이트 정상 작동 확인
- **SEO 콘텐츠 에이전트**: 평일 저녁 7시 15분, "오늘의 시황" 글 자동 작성·게시

지금은 실제 사이트 주소가 없어 "건너뜀" 알림만 오도록 되어 있습니다. **2단계(배포)와 5단계(ADMIN_TOKEN 설정)가 끝나면**, 이 대화로 돌아와 다음과 같이 말씀해주세요:

> "사이트 주소는 https://www.내도메인.com 이고 ADMIN_TOKEN은 [Render에 입력한 값]이야. 예약된 에이전트 3개에 반영해줘"

그러면 3개의 예약 작업 프롬프트에 실제 주소와 토큰을 채워 넣어 바로 작동하도록 해드립니다.

## 8. 홍보 방법

사이트가 열렸다면 사람들이 찾아오게 만들어야 합니다.

**검색엔진 유입 (SEO)**
- 오늘의 시황 콘텐츠가 꾸준히 쌓이면 "삼성전자 유사종목", "종목 조합 추천" 같은 검색어로 자연 유입이 생깁니다. 시간이 걸리는 방법이지만 가장 꾸준한 트래픽 소스입니다.
- 네이버 서치어드바이저(searchadvisor.naver.com), 구글 서치콘솔(search.google.com/search-console)에 사이트를 등록하면 검색 노출이 빨라집니다.

**커뮤니티**
- 디시인사이드 주식 갤러리, 네이버 종목토론방, 각종 주식 오픈카톡방 등에 "이런 사이트 만들었는데 써보세요" 식으로 자연스럽게 소개 (과도한 광고성 게시물은 커뮤니티 규칙 위반이 될 수 있으니 후기/정보공유 형태로)
- 네이버 카페(주식 관련 대형 카페들) 가입 후 정보 공유 게시판에 소개

**SNS**
- 매일 자동으로 쌓이는 "오늘의 시황" 콘텐츠를 그대로 캡처하거나 요약해서 스레드(Threads), 인스타그램, X(트위터)에 매일 올리기 — 이미 콘텐츠가 자동 생성되니 재활용만 하면 됩니다
- 유튜브 쇼츠/틱톡으로 "오늘 이 종목 이렇게 움직였어요" 식의 15초 영상 콘텐츠도 효과적

**초기 트래픽 팁**
- 처음엔 가족·지인에게 링크를 공유해 실제 사용 피드백을 받아보세요
- 사이트 하단에 "이 사이트가 도움이 됐다면 공유해주세요" 문구를 넣는 것도 좋습니다
